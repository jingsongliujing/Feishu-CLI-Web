# Natural Language Recipes

这些是适合直接交给 AI / Cursor / Claude 的飞书 CLI 自然语言模板。表达越清晰，命中率越高。

## IM 消息

- 帮我给黄云发飞书消息：今天请提交周报
- 帮我给【项目沟通群】发消息：今天 18:00 统一发版，请大家自测
- 帮我在【产品群】里搜索关键词：接口变更
- 帮我把【项目沟通群】最近 3 条群消息整理成摘要

推荐串联：

```text
搜索用户 -> 获取 open_id -> 发送私聊消息
搜索群聊 -> 获取 chat_id -> 搜索群消息 / 发送群消息 / 读取群消息
```

## 日历

- 帮我查看我今天的日程
- 帮我查看我这周的日程
- 帮我创建明天下午 3 点的需求评审会，时长 1 小时，邀请张三、李四
- 帮我查一下张三明天是否有空
- 帮我快速创建一个 30 分钟的临时会议

推荐串联：

```text
搜索参会人 -> 获取 open_id -> calendar +create
读取 agenda / freebusy -> 汇总空闲时间 -> 返回建议
```

## 云文档

- 帮我新建一篇名为【周报 - 2026-04-15】的文档
- 帮我新建一篇文档，内容用 Markdown 写本周工作小结
- 帮我读取文档 docxxxxxx 的内容并总结
- 帮我把当前目录的 report.md 上传为飞书文档

推荐串联：

```text
docs +create
docs +fetch -> 总结
读取本地 markdown -> docs +create
```

## 多维表格 Base

- 帮我列出我有权限的多维表格
- 帮我在【任务管理表】里新增一条记录：任务 = 修复登录 bug，负责人 = 张三
- 帮我查询【销售数据表】中状态为进行中的记录
- 帮我把多维表格数据导出成 CSV

推荐串联：

```text
定位 base / table -> 查询字段或记录 -> 新增 / 搜索 / 导出
涉及负责人时先搜索联系人拿 open_id
```

## 任务

- 帮我查看我今天待办任务
- 帮我查看我已完成的任务
- 帮我新建一个任务：明天 12 点前完成接口文档
- 帮我把任务【XXX】标记为已完成

推荐串联：

```text
task +get-my-tasks
task +create
按任务名搜索 -> 获取 task_id -> task +complete
```

## 通讯录

- 帮我搜索同事：张三
- 帮我查看部门成员列表
- 帮我获取张三的飞书邮箱 / 手机号

推荐串联：

```text
contact +search-user
contact +get-user
```

## AI 自动办公

- 帮我汇总今天完成的任务，生成周报并保存为文档
- 帮我整理今天会议内容，生成会议纪要并发到群里
- 帮我总结项目群今天的所有重要消息
- 帮我把本周日程整理成工作周报

推荐串联：

```text
读取任务 / 日程 / 群消息 -> 总结 -> docs +create / im +messages-send
```

## 通用模板

```text
帮我【操作】【对象】【内容 / 条件】
帮我 发消息 给 黄云 内容是 今天请提交周报
帮我 发消息 到 【技术群】 内容是 今天 18:00 发版
帮我 搜索 【产品群】 里的关键词：接口变更
帮我 新建文档 标题 【周报】 内容 本周完成了飞书CLI开发功能
帮我 查询多维表格【销售数据表】里状态=进行中的记录
```
